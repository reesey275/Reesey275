# DevonBoarder

## Problem Statement
Accurate, accessible maps of the Devon coastline are limited.
This makes it difficult for visitors and researchers to explore coastal paths and points of interest.

## Approach
An interactive web map built with Leaflet and OpenStreetMap provides detailed coastal data through a Node.js backend.

## Role
Lead developer.

## Tech Stack
Leaflet, OpenStreetMap, Node.js.

## Outcomes
Delivered a proof-of-concept map that supports coastal navigation and planning for future conservation work.

## Next Steps
Integrate real-time environmental data and optimize the interface for mobile use.
